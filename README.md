## 📌 Task 3 – Tic-Tac-Toe Web Application  
Create an interactive Tic-Tac-Toe game using HTML, CSS, and JavaScript.  
The game should allow users to play against each other or against an AI.  

## 🚀 Features  
- 2-player mode  
- AI opponent option  
- Winning condition checks  
- Responsive UI  

## 🛠️ Tech Stack  
- HTML  
- CSS  
- JavaScrip
